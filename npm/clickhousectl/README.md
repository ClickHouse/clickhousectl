# clickhousectl

The CLI for ClickHouse: local and cloud.

## Install

    npm install -g @clickhouse/clickhousectl
    # or
    npx @clickhouse/clickhousectl local install stable

Supported platforms: macOS (arm64, x86_64), Linux (arm64, x86_64).

The `chctl` alias is installed alongside `clickhousectl`.

See the [full README](https://github.com/ClickHouse/clickhousectl#readme) for usage.
